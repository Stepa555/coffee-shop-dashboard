# ☕ Панель продаж кофейни — Портфолио Junior Data Analyst

**Роль:** Junior Data Analyst  
**Инструмент:** Power BI  
**Период:** январь – июнь 2023  
**Масштаб:** 214 тыс. заказов, 3 магазина, выручка 699 тыс.

---

## 📊 О проекте

Дашборд анализирует продажи сети кофеен из 3 точек. Цель — выявить точки роста и предложить конкретные бизнес-рекомендации.

### Структура дашборда

| Страница | Содержание |
|----------|------------|
| 1 | KPI, тренды по месяцам, дням недели, категориям |
| 2 | Детализация по продуктам (SKU) и часовая динамика |
| 3 | Анализ по магазинам + структура категорий |
| 4 | Выводы и рекомендации |

---

## 🔍 Ключевые инсайты

- **Утренний пик (6–9 утра)** — 92 тыс. выручки (13% от общего объёма)
- **Coffee beans (кофе в зёрнах)** — средний чек ~38 при всего 787 заказах (высокая маржинальность)
- **Суббота** — на 5% ниже будних дней
- **Coffee** даёт 39% выручки, **Tea** — 10%

---

## 💡 Рекомендации

| № | Рекомендация | Ожидаемый эффект |
|---|--------------|------------------|
| 1 | Усилить утреннюю смену (6–10) + комбо «кофе + выпечка» | +5–7% выручки |
| 2 | Продвигать Coffee beans через кассу | +20–30% в сегменте |
| 3 | Субботняя акция на выпечку | +3–5% по субботам |

**Прогнозируемый рост выручки:** +8–12%
